# 📅 VidaOrganizada CLI

![Versão](https://img.shields.io/badge/Versão-1.0.0-blue)
![Autor](https://img.shields.io/badge/Autor-Kawe_Lobo-orange)

Um sistema simples e direto em linha de comando (CLI) construído em Python para ajudar você a gerenciar suas tarefas.

---

## 🎯 O Problema
A dificuldade e a sobrecarga de tentar organizar as tarefas do dia a dia sem uma ferramenta rápida e acessível.

## 💡 A Solução
Um sistema minimalista em Python que permite adicionar, organizar e listar suas tarefas diretamente do seu terminal, sem distrações.

---

## 🚀 Como usar

Para executar a aplicação, abra o seu terminal na raiz do projeto e rode o seguinte comando:

```bash
python src/main.py
